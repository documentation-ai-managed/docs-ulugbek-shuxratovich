## Webhook Setup

Set up webhooks to trigger actions on events like page creation or updates.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to Workspace Settings > Integrations > Webhooks. Click "New Webhook".
  </Step>
  <Step title="Configure Events" icon="settings">
    Select events such as `page.created` or `page.updated`. Add your endpoint URL.
  </Step>
  <Step title="Test and Save" icon="play">
    Send a test payload. Verify receipt at your server, then save.
  </Step>
</Steps>

<Request tabs="cURL" show-lines="true">
```bash
curl -X POST https://api.example.com/v1/webhooks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://your-webhook-url.com/webhook",
    "events": ["page.created", "page.updated"]
  }'
```
</Request>

## API Access for Extensions

Build custom extensions using the REST API. Authenticate with `YOUR_API_KEY`.

<ParamField path="pageId" param-type="string" required="true">
  Unique page identifier.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Bearer YOUR_API_KEY`.
</ParamField>

<CodeGroup tabs="JavaScript,Python">
```javascript
const response = await fetch('https://api.example.com/v1/pages', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});
const pages = await response.json();
console.log(pages);
```

```python
import requests
response = requests.get(
    'https://api.example.com/v1/pages',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)
pages = response.json()
print(pages)
```
</CodeGroup>
