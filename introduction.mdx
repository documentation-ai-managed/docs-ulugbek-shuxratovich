## Overview

Ulugbek Shuxratovich provides a centralized documentation space where you organize, manage, and share all your project documentation. Create pages, embed code examples, and collaborate seamlessly with your team. Customize themes and layouts to match your brand, ensuring consistency across all docs.

This space supports rich MDX components for interactive guides, API references, and changelogs. Start by exploring key features below.

## Quick Start

Get up and running in minutes. Follow these steps to create your first documentation page.

<Steps>
  <Step title="Create Account" icon="user-plus">

    Sign up at `https://ulugbekshuxratovich.com/register` with your email.

  </Step>
  <Step title="New Workspace" icon="plus">

    Click **New Workspace** and name it after your project, like `my-project-docs`.

  </Step>
  <Step title="Add First Page" icon="file-plus">

    Select **New Page**, choose `introduction.mdx`, and start editing with MDX syntax.

    <CodeGroup tabs="CLI,Web">
      ```bash
      npx create-ulugbek@latest my-docs
      cd my-docs
      npm run dev
      ```
      ```typescript
      // Embed in your app
      import { UlugbekDocs } from '@ulugbek/docs';
      <UlugbekDocs workspaceId="your-workspace-id" />
      ```
    </CodeGroup>

  </Step>
</Steps>

<Callout kind="tip">
  Pro tip: Use the preview pane to see changes live as you edit.
</Callout>
